Ai no neko

En se basant sur le concept des “Jam”, le principe du concours “Code Game Jam” est de créer un jeu vidéo dans un temps donné, tout en respectant le thème de l’édition. Les participants ont alors 30 heures pour pouvoir développer un petit jeu vidéo pour lequel ils ont cartes blanches autant d’un point de vue technique (langage de programmation, logiciel utilisés, format du jeu) que graphique (moteur de jeu, boîtes à outils).

Créativité, collaboration et fun sont les maîtres mots du concours Code Game Jam ! Ceci est la Code Game Jam 2023 lors de ma seconde année d'étude en Informatique. Nous étions dans une équipe de 5.

Le thème de notre projet était “Légende Éthérée”, un sujet qui a stimulé notre imagination et nous a poussés à repousser les limites de notre créativité.

Ce projet a été une expérience extrêmement enrichissante, nous permettant d'explorer de nouvelles technologies, de travailler en équipe et de développer nos compétences de conception et de développement en HTML, CSS, mais surtout en JavaScript.

Nous sommes fiers du travail accompli lors de cette Code Game Jam et nous sommes heureux de pouvoir partager cette expérience avec vous à travers notre portfolio.

Compétences

    Réaliser un développement d’application / Optimiser des applications informatiques: J'ai utilisé PHP, HTML, CSS et JavaScript pour mettre en place une architecture robuste et assurer le bon fonctionnement de l'application web. Cette collaboration m'a permis d'améliorer mes compétences relationnelles et ma capacité à travailler en équipe, renforçant ainsi mon expérience de travail dans un environnement informatique collaboratif.
